Trait capturing a traversal of a trie.

By providing functions for each of the different cases, it is possible to describe a number of different traversals. For now it's probably best to view the source for run to understand how best to implement each function.

Associated Types

type Key: 'a

Key type to be threaded through by run, needn't be K (is often &'a K).

type Value: 'a

Value type to be threaded through by run, needn't be V (is often ()).

Type returned by the entire traversal, for insert it's Option<V>.

Run the traversal, returning the result.

Let key_fragments be the bits of the key which are valid for insertion below the current node such that the 0th element of key_fragments describes the bucket that this key would be inserted into.
